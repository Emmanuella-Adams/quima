# Quima: Quantum 3D Playground & Simulation Lab

Designed & Engineered with ❤️ by **Emmanuella Adams**.

Quima is an interactive, highly visual digital universe designed to build intuition for fundamental concepts of quantum mechanics. Combining physics equations, interactive 3D simulations, and gamified challenges, Quima provides a sandbox environment to master quantum mechanics in a beautiful, dark cosmic atmosphere.

---

## 🌌 Key Highlights & Features

### 1. Quantum Telemetry Stream (Event Log Panel)
Underneath the main simulation, a real-time **Quantum Telemetry Stream** displays a chronological timeline of quantum state transitions and physical measurement operations. Real-time logging events trigger based on:
* **Time-Evolution Status**: Starting and pausing the Schrödinger equation evolution.
* **Measurement Collapses**: Squeezing probabilistic clouds into physical coordinates with exact output tuples.
* **Property Modifications**: Spatial boundaries ($dx$), potential barriers ($V_0$), and external magnetospheres.
* **Postulate Achievements**: Immediate recognition when interactive challenges are met.

### 2. Live Mathematical Theory Deck
An interactive sliding menu that pairs complex quantum equations with the corresponding simulation sliders. Toggle concepts in real-time to preview calculations for:
* **Born Probability Waves** ($P(x) = |\Psi(x)|^2 dx$)
* **Heisenberg Uncertainty Relation** ($\Delta x \cdot \Delta p \geq \frac{\hbar}{2}$)
* **EPR Entanglement Links** ($|\Psi_{Bell}\rangle = \frac{1}{\sqrt{2}}(|\uparrow\downarrow\rangle - |\downarrow\uparrow\rangle)$)
* **Potential Tunneling Transmission rates** ($T \approx e^{-2 w \kappa}$)
* **Larmor Magnetic Precession** ($\Omega = -\gamma B_0$)

### 3. Five Simulated Physical Core Systems
Interactive, beautiful 3D particle landscapes built on Top of Three.js and custom math models:
* **Wavefunction Grid**: Double-excited superposition waves mapping real-time wavefunction densities.
* **Uncertainty Stage**: Constraining a particle vertically to observe instant momentum energy fluctuations.
* **Bell Entanglement Pairs**: Live monitoring of Spin-Up ($\uparrow$) and Spin-Down ($\downarrow$) spin locks.
* **Barrier Tunneling Layer**: Leaking waves that experience exponential decay when striking customizable potential walls.
* **Spin Precession Spheres**: Dual orbital gyromagnetic spheres tracing precession cones in a vector field.

### 4. Quantum Experimental Challenges
* **Coordinate Estimation**: Predicting standard deviations and capturing high density coordinate envelopes.
* **Wave Stabilization**: Adjusting wavelength parameters to synchronize complex sine wave nodes.
* **Tunneling Maximization**: Optimizing barrier width and coordinate energy states to pass through the wall.
* **State Coherence Link**: Matching Alice's spin with phase states to guarantee perfect EPR telemetry coherence.

## 🛠️ Technology Stack
* **UI & Rendering**: React 18, Vite, Three.js (WebGL Canvas), HTML5 Canvas.
* **Transitions**: `motion` layout and micro-transitions.
* **Design & Styling**: Tailwind CSS, custom Lucide vectors, and high-contrast glowing UI components.
* **Typography**: Clean *Space Grotesk* for UI interfaces and *JetBrains Mono* for telemetry tables and precision figures.

## 🚀 Running locally

### 1. Install Dependencies
```bash
npm install
```

### 2. Start Development Server
```bash
npm run dev
```

### 3. Production Build
```bash
npm run build
```

The compiled application assets will be prepared under `dist/`.
